# 戰情室會議系統 v1.0

## 核心
不是「一個 AI 給一個答案」,是「7 個 agents 開會給建議」。

## 流程
1. 使用者問投資問題
2. 呱呱召集會議
3. 各 agent 依流派發言(基本/技術/籌碼/量化/逆向/風控)
4. 呱呱整合結論

## 範例
使用者:「我想買 2330」
→ 評級師:PE 合理
→ 技術師:短期風險
→ 籌碼家:外資買超
→ 科學家:勝率 58%
→ 質疑官:你確定?
→ 風控師:停損 7% 部位 15%
→ 呱呱整合:可以買,分批進場

## 技術
- Backend:依序呼叫每個 agent API(不同 system prompt)
- Frontend:戰情室畫面,streaming 發言
- 成本:7 次呼叫 ≈ $0.05-0.15/次
- 訂閱費 > 成本即獲利
