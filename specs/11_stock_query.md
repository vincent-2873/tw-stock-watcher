# 🔍 股票查詢功能規格書

> Vincent 可以隨時指定查詢任何一檔股票,系統回傳完整分析

---

## 🎯 使用情境

**Vincent 想法:**
> 「我突然想研究某一檔股票(例如看到朋友推薦、或新聞提到),
>  我希望打開系統搜尋它,立刻拿到完整的分析」

**關鍵要求:**
- 打字少(手機通勤時用)
- 回應快(< 3 秒看到結果)
- 資訊分層(先摘要,再細節)
- 可深度挖掘(需要時)

---

## 🧠 兩種查詢模式

### 模式 A:**快速查詢**(預設,< 3 秒)
- 使用快取資料
- 不重新呼叫 AI
- 適合「我想知道現在什麼情況」
- 若快取 < 5 分鐘 → 秒開

### 模式 B:**深度分析**(15-30 秒)
- 重新抓最新資料
- 呼叫 Claude API 重新分析
- 適合「我要認真研究要不要買」
- Vincent 需手動點「深度分析」按鈕

## 🔧 關鍵實作細節

### 股票識別

```python
async def resolve_stock_query(query: str) -> dict:
    """
    把使用者輸入解析成標準股票代號
    
    支援:
    - "2317" → 台股 2317
    - "鴻海" → 台股 2317
    - "AAPL" → 美股 AAPL
    - "蘋果" → 美股 AAPL
    - "tsmc" → 需問清楚(可能是 2330 或 TSM)
    """
    query = query.strip()
    
    # 1. 全數字 → 台股
    if query.isdigit() and len(query) == 4:
        return await lookup_tw_stock(query)
    
    # 2. 全英文大寫 → 美股
    if query.isupper() and query.isalpha():
        return await lookup_us_stock(query)
    
    # 3. 中文 → 先查台股
    if contains_chinese(query):
        candidates = await fuzzy_search_tw(query)
        if len(candidates) == 1:
            return candidates[0]
        return {"need_choice": True, "candidates": candidates}
    
    # 4. 混合 → 跨市場搜尋
    candidates = await multi_market_search(query)
    if len(candidates) == 1:
        return candidates[0]
    
    return {"need_choice": True, "candidates": candidates}
```

async def fetch_all_stock_data(stock_id: str) -> dict:
    """
    並行抓所有資料

    async with asyncio.TaskGroup() as tg:

### API Endpoint

```python
# routes/stocks.py

@router.get("/api/stocks/{query}")
async def get_stock_analysis(
    query: str,
    mode: str = "quick",  # "quick" | "deep"
    include_social: bool = True,
):
    """
    統一入口 - 查詢單一股票
    """
    # Step 1: 識別
    stock = await resolve_stock_query(query)
    if stock.get("need_choice"):
        return {"status": "need_choice", "candidates": stock["candidates"]}
    
    # Step 2: 快取
    if mode == "quick":
        cached = await get_cached_analysis(stock["id"], max_age=300)
        if cached:
            return cached
    
    # Step 3-6: 並行抓資料
    data = await fetch_all_stock_data(stock["id"])
    
    # Step 7: 評分
    analysis = await run_decision_engine(data)
    
    # Step 8: AI 深度(可選)
    if mode == "deep":
        analysis["ai_insights"] = await ai_deep_analysis(data)
    
    # Step 9: 儲存
    await save_query_record(stock["id"], analysis)
    
    return analysis
```
