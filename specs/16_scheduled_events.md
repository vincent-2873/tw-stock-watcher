# 📅 重要日程日曆規格書

> Vincent 不知道「明天有法說會」就是致命盲點。
> 這份規格解決所有「時間點敏感」的投資事件。

## 🎯 核心原則

**系統必須知道:**

```
未來 30 天內,每一檔自選股會發生什麼重要事件?

未來 3 個月內,哪些全市場重要日子要避開?

過去 30 天內,發生過什麼影響至今的事件?
```

**沒掌握這些時間點 → 推薦建立在沙灘上。**

## 📋 必追蹤的 10 類事件

### 1. **法說會**(最重要)

台灣上市公司法說會:
- 季報後約 1 個月內
- 重大事件後召開
- **提前 1-2 週公告**

**資料來源:**
- 公開資訊觀測站「最新法說會公告」
  - URL: `https://mops.twse.com.tw/mops/web/t100sb02_1`
- TWSE 重大訊息
- 各公司投資人關係網頁

### 2. **財報公布日**

**台股固定節奏:**

| 財報類型 | 公布期限 | Vincent 要注意 |
|---------|---------|--------------|
| 年報 | 隔年 3/31 前 | 大部分公司 3 月底 |
| Q1 財報 | 5/15 前 | 金融業 5/30 前 |
| Q2 財報 | 8/14 前 | |
| Q3 財報 | 11/14 前 | |
| 月營收 | 每月 10 日前 | **最重要,反應快** |

**公布時間:**
- 盤後公布(13:30 後)→ 次日反應
- 少數盤中公布 → 即時反應

### 3. **除權除息日**

**影響:**
- 除息日股價會「跳空」下跌(扣除股利)
- **股利 > 5%** 的股票,跳空幅度大
- 填息 / 貼息是重要指標

**時程:**
- 股利分派案:股東會通過(通常 6 月)
- 除息日:7-9 月居多
- 除權日:一般在除息前後

### 4. **股東會**

**影響:**
- **停過戶期**:股東會前 60-30 天,股權定格
- 股東會當日:有新政策 / 分派方案 → 股價反應
- 公司派 vs 市場派爭奪:戲劇化影響

### 5. **停資停券**(⚠️ 散戶陷阱)

**為什麼重要:**
```
停資:融資戶不能買進 → 買盤減少
停券:融券戶不能放空 → 空方壓力減小
停資停券:通常配合重大事件(如股東會、除息)

時機:
 - 除息前後 6 個交易日
 - 股東會前後停過戶期
 - 處置股
```

**實戰影響:**
- 停券前**融券需回補** → 強制買進 → 股價常飆
- 例如「融券回補行情」是業界熟知的操作

### 6. **處置股 / 全額交割 / 變更交易**

**處置股**:連續漲停/跌停超過一定比例
- **第 1 次處置**:人工盤 + 預先收足款券
- **第 2 次處置**:全額交割

**信號:**
- 處置股通常是「炒作被抓」
- 散戶追高後常遇到處置 → 流動性變差
- Vincent 要避開

### 7. **融資融券異常**

**需警戒的指標:**
```
- 融資餘額單日增 > 5%:散戶湧入(短期反指標)
- 券資比 > 30%:做空壓力大
- 融資維持率 < 130%:斷頭潮風險
- 借券賣出餘額暴增:外資可能做空
```

### 8. **MSCI / FTSE 指數調整**

**時程:**
- MSCI:每年 2、5、8、11 月中下旬
- FTSE:每年 3、6、9、12 月

**影響:**
- 被新納入 → 被動基金買進 → 股價漲
- 被剔除 → 被動基金賣出 → 股價跌
- **調整生效前 5-10 天**:股價先反應

### 9. **期貨選擇權結算日**

**台指期:**
- 每月**第 3 個週三**結算
- 結算日前後波動大
- 「結算日行情」是業界現象

**台指選擇權:**
- 週選:每週三結算
- 月選:配合期貨

### 10. **台灣國定假日 / 補班**

**影響:**
- 連假前:散戶出場觀望 → 成交量縮
- 連假後:補漲或補跌
- **補班日不是交易日**(周六不開盤)

## 🔧 技術實作

### 資料庫 schema

```sql
CREATE TABLE scheduled_events (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    
    -- 事件分類
    event_type VARCHAR(50) NOT NULL,
    -- 'earnings_call', 'financial_report', 'ex_dividend', 'ex_rights',
    -- 'shareholders_meeting', 'margin_restriction', 'disciplinary_stock',
    -- 'msci_review', 'futures_settlement', 'holiday',
    -- 'us_earnings', 'us_fomc', 'us_economic_data', 'us_holiday'
    
    -- 時間(嚴格區分時區!)
    event_date DATE NOT NULL,
    event_time_tpe TIMESTAMPTZ,         -- 台北時間
    event_time_et TIMESTAMPTZ,          -- 美東時間(若為美股事件)
    
    -- 關聯股票(若有)
    stock_id VARCHAR(10),
    stock_name VARCHAR(100),
    market VARCHAR(20),  -- 'TW' / 'US'
    
    -- 事件內容
    title VARCHAR(200) NOT NULL,
    description TEXT,
    details JSONB,
    
    -- 影響評估
    estimated_impact INT,  -- 1-10
    affects_stocks TEXT[],  -- 會影響的其他股票
    
    -- 資料來源
    source_url TEXT,
    source_verified_at TIMESTAMPTZ,
    
    -- 提醒
    remind_days_before INT[],  -- [7, 3, 1] 提前幾天提醒
    reminded_at TIMESTAMPTZ[],  -- 已提醒過的日期
    
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_events_date ON scheduled_events(event_date);
CREATE INDEX idx_events_stock ON scheduled_events(stock_id);
CREATE INDEX idx_events_type ON scheduled_events(event_type);
```
