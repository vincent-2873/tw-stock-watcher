> 給前端開發者的完整設計指南

## 設計哲學

### 三個核心原則

**1. 資訊分層(最重要!)**
- 首屏 = 結論
- 第二屏 = 理由
- 第三屏 = 原始資料
- **不要一次塞所有資訊,讓使用者選擇深入**

**2. 手機優先**
- Vincent 80% 時間用手機看
- 設計先做 Mobile(375px)
- 再 Tablet(768px)
- 最後 Desktop(1280px+)

**3. 行動導向**
- 看完報告應該知道「現在要做什麼」
- 每頁都有「下一步 CTA」
- 避免「只能看、不能做」的被動 UI

## 色彩系統

### 語意色

```css
/* 漲跌色 - 台股習慣(紅漲綠跌) */
--up: #e74c3c;        /* 紅色 - 上漲 */
--down: #27ae60;      /* 綠色 - 下跌 */
--flat: #6b7280;      /* 灰色 - 持平 */

/* 注意:美股習慣是綠漲紅跌,要設定「市場模式」切換 */

/* 功能色 */
--primary: #3b82f6;   /* 藍 - 主要動作 */

--warning: #f59e0b;   /* 橘 - 警告 */
--danger: #ef4444;    /* 紅 - 危險 */

/* 背景色 */

--bg-secondary: #f9fafb;

/* 暗模式 */
```

## 互動模式設計

### Pull-to-Refresh(手機)
- 下拉重新載入
- 所有列表頁都有

### Swipe 手勢
- 自選股卡片:左滑出現「移除」
- 警報:左滑出現「標為已處理」

### 長按操作
- 股票卡片長按 → 快速操作選單

### Loading 狀態
- 骨架屏(Skeleton)
- 不用 Spinner(過時)

### Empty 狀態
- 圖示 + 簡短說明 + 引導動作
- 例:「還沒加自選股,先找一檔吧 →」

### Error 狀態
- 友善的錯誤訊息
- 「稍後再試」按鈕
- 不用技術術語

## 動畫規範

**原則:快速、有意義、不打擾**

```css
/* 預設過渡 */
--transition-fast: 150ms ease;
--transition-base: 250ms ease;
--transition-slow: 500ms ease;

/* 不建議使用 */
- Bouncy 動畫(看起來不專業)
- 旋轉動畫(金融產品要沉穩)
- 超過 1 秒的動畫
```

**常用動畫:**
- 頁面進入:fade-in 200ms
- 卡片出現:slide-up 300ms
- 數字變動:count-up 500ms
- 推薦結論:逐字打字效果(僅首次)

---

## 資料視覺化

### K 線圖
- 使用 **TradingView Lightweight Charts**(免費)
- 支援多時間框架切換
- 可加技術指標

### 長條圖、折線圖
- 使用 **Recharts**(React 生態)
- 簡單、響應式

### 特殊圖表
- 桑基圖(資金流向):用 **@nivo/sankey**
- 熱力圖(產業輪動):用 **Recharts Treemap**
- 關係圖(供應鏈):用 **React Flow** 或 **Cytoscape.js**

---

## 無障礙 (A11y)

- 所有互動元素有 aria-label
- 色彩對比符合 WCAG AA
- 鍵盤可操作(Tab 順序正確)
- 螢幕閱讀器友善

---

## 效能要求

- 首頁 LCP < 2.5s
- 頁面切換 < 500ms
- 資料更新 < 1s
- 使用 React Server Components(RSC)
- 圖表 lazy load

## 完整元件清單(讓前端照做)

```
基礎元件:
├─ Button (primary, secondary, danger, ghost)
├─ Input (text, number, search)
├─ Select
├─ Switch
├─ Checkbox
├─ Radio
├─ Badge
├─ Card
├─ Modal / Dialog
├─ Drawer (手機下方拉起)
├─ Toast / Notification
├─ Tabs
├─ Accordion
└─ Skeleton Loader

金融專用元件:
├─ PriceDisplay (含漲跌色)
├─ ChangeIndicator (+/- 符號動畫)
├─ ConfidenceMeter (信心度條)
├─ ScoreBar (四象限評分)
├─ StockCard (自選股卡)
├─ KLineChart (K 線)
├─ VolumeChart (量能)
├─ HeatMap (熱力圖)
├─ SankeyFlow (桑基圖)
├─ AlertBadge (警報)
├─ RecommendationCard (推薦卡)
└─ PerformanceIndicator (績效指標)
```
