## 📋 首頁要呈現的 11 個區塊（按順序）

### 1️⃣ 頂部導航
- 🦆 呱呱投資招待所 logo + Quack House 英文副標
- 右側導航：🏠 今日 / 🔥 題材 / 📊 自選 / 🗺️ 產業 / ⋯ 更多
- 「更多」點開：回測、模擬、警示、報告、設定

### 2️⃣ 問候區（greeting）
- 「早安/午安/晚安，Vincent」（根據時段變化）
- 副標：呱呱一句話（根據市場狀態變化）
- 右側：日期 + 即時狀態（盤前/盤中/盤後）

### 3️⃣ 🌡️ 今日市場脈動（四欄）
- 加權指數 TAIEX
- 台指期
- 費城半導體
- VIX 恐慌指數
- 每格都有：當前值、漲跌、附註（量、警示）

### 4️⃣ 🦆 呱呱今日功課（核心卡片）
- 呱呱 emoji + 時間標籤
- 主文案（一句話總結今天市場）
- 💡 補漲機會（3 檔）
- ⚠️ 避免追高（3 檔）
- 3 個行動按鈕

### 5️⃣ 🔥 今日題材熱度（左欄）
- 最多 5 個題材卡
- 每張卡包含：排名、題材名、新/熱徽章、熱度分數、趨勢箭頭、熱度條、描述、相關個股

### 6️⃣ 🌏 昨夜美股連動（右欄上）
- 4 個美股事件
- 每個事件：標題、→ 影響台股、相關個股

### 7️⃣ 💎 今日焦點股（右欄下）
- 5 檔焦點股
- 每個：股名、所屬題材、現價、漲跌

### 8️⃣ 🏔️ 供應鏈金字塔（CCL 為例）
- 5 層金字塔：終端 → PCB → CCL → 樹脂 → 玻纖
- 每層的狀態：🔴🟡🟢🔥
- 「補漲層」特別標紅
- 圖例說明

### 9️⃣ 💰 三大法人資金流向（三欄）
- 外資 / 投信 / 自營商
- 每欄：金額、5 日對比、狀態徽章、前 3 買/賣超

### 🔟 🗺️ 產業熱力圖（2×4 格）
- 8 個主要產業
- 顏色表示熱度：紅熱（流入）、綠冷（流出）
- 每格：產業名、漲跌%、資金量

### 1️⃣1️⃣ 🔄 產業資金輪動（兩欄）
- 左：流入 TOP 5
- 右：流出 TOP 5
- 每個：產業名、金額、視覺化條
- 底部：呱呱解讀一句話

### 1️⃣2️⃣ 空狀態（自選股、警示）
- **不要再用「—— 尚無 ——」**
- 要用引導性設計：
  - 圖示（🦆💤 或 🔕）
  - 標題（筆記本還是空白的）
  - 副標（呱呱幫你準備了幾個開始的方式）
  - 3 個行動按鈕

### 1️⃣3️⃣ 🦆 右下角浮動呱呱按鈕
- 所有頁面都要有
- 64px 圓形，暗金色
- 有紅點通知（如果有新事件）
- 點擊開啟對話框

## 🎨 視覺設計要點

### 配色（嚴格執行）

```css
背景色
--bg-primary: #1A1815       /* 主背景（深炭灰）*/
--bg-card: #26231C          /* 卡片背景 */
--bg-elevated: #302C22      /* 浮起元素 */

文字色
--text-primary: #E8E3D5     /* 主文字（米白）*/
--text-secondary: #A89F8A   /* 次要 */
--text-muted: #6B6353       /* 弱提示 */

重點色
--accent-gold: #C9A961      /* 暗金（最重要！品牌色）*/
--accent-moss: #7A8471      /* 墨綠 */

台股漲跌（侘寂風，不要俗豔紅綠）
--rise: #C9754D             /* 暗紅銅（漲）*/
--fall: #6B8E5A             /* 暗苔綠（跌）*/
```

### 字體

```css
中文標題：Noto Serif TC（思源宋體，有文化感）
中文內文：Noto Sans TC（思源黑體）
英文副標：Cormorant Garamond（古典感）
數字：JetBrains Mono（等寬，金融感）
```

### 間距

**嚴守 8px grid**：4, 8, 12, 16, 20, 24, 32, 48

### 圓角

- 小元素：2px（侘寂風不用大圓角）
- 卡片：6-8px
- 浮動按鈕：50%（只有呱呱是圓的）
