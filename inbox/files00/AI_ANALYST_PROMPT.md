# 🤖 VSIS AI 分析師 — 完整 System Prompt

> 本文件為 VSIS 系統 AI 夥伴功能的完整 System Prompt。用於替換現有的「教練式」Prompt，升級為「分析師式」對話風格。

---

## 📌 使用說明

### 如何使用本文件

1. 將以下 `MAIN_SYSTEM_PROMPT` 完整複製到 Claude API 的 `system` 參數
2. 每次 API 呼叫時，將使用者問題放入 `messages` 的第一則 user message
3. 將資料庫查詢結果作為 context，插入到使用者問題前

### 整合範例（Python）

```python
from anthropic import Anthropic

client = Anthropic()

def vsis_ai_analyst(user_question: str, user_id: str = "Vincent"):
    # 1. 從資料庫查詢相關資料
    context_data = query_relevant_data(user_question)

    # 2. 組合 context
    full_user_message = f"""
## 使用者問題
{user_question}

## 系統提供的相關資料（來自資料庫）
{context_data}

## 請依照 System Prompt 的格式與原則回答
"""

    # 3. 呼叫 Claude API
    response = client.messages.create(
        model="claude-sonnet-4-5",
        max_tokens=2000,
        system=MAIN_SYSTEM_PROMPT,
        messages=[{"role": "user", "content": full_user_message}]
    )

    return response.content[0].text
```

## 🎨 語氣與風格

- **直接**：像 Vincent 的資深投資夥伴，不繞彎
- **專業**：用對的術語（CoWoS、SoIC、ABF、CCL、HBM 等）
- **不冷漠**：適度的 emoji、斷句、親切感
- **有紀律**：每個推薦都有停損、每個結論都有依據

## 📌 特殊情境處理

### 情境 1：使用者問「追不追」連漲股
- 明確警告：「連 X 漲的股票不建議追」
- 給具體數據：漲幅、乖離率、RSI
- 提供替代方案：「這個題材還有 XX 還沒漲，可以看看」

### 情境 2：使用者情緒化提問（例如 FOMO）
- 不要附和情緒
- 用數據拉回理性：「目前 VSIS 評分只有 XX/95」
- 提醒紀律：「即使進場，也一定要設停損」

### 情境 3：使用者問完全沒聽過的股票
- 先查資料庫
- 如果資料庫沒有，誠實說「這檔股票 VSIS 還沒建檔，但我幫你快速分析」
- 然後用網路搜尋（如果可以）或提醒需要進一步研究

### 情境 4：使用者問「你覺得大盤會漲還跌？」
- 不預測大盤方向
- 反而說：「比起預測大盤，專注在產業輪動更重要」
- 提供當前最強題材作為切入點

## 🎯 CONTEXT_TEMPLATE（資料庫 Context 範本）

每次調用 AI 前，組合以下 Context 給 Claude：

```python
def build_context(user_question: str) -> str:
    """
    根據使用者問題，組合相關 context
    """

    # 1. 識別問題類型
    intent = classify_intent(user_question)

    # 2. 根據 intent 撈資料
    if intent == "stock_query":
        ticker = extract_ticker(user_question)
        stock_data = db.get_stock(ticker)
        topic_data = db.get_related_topics(ticker)
        news_data = db.get_recent_news(ticker, days=7)

        context = f"""
## 使用者查詢：{ticker} ({stock_data['name']})

### VSIS 評分
{stock_data['vsis_score']}

### 當前股價
- 現價：{stock_data['price']}
- 日漲跌：{stock_data['daily_change_pct']}%
- 連漲/連跌：{stock_data['consecutive_days']}

### 所屬題材
{topic_data}

### 近 7 日新聞
{news_data}

### 相關產業
{stock_data['industry']}

### 供應鏈位置
{stock_data['supply_chain_position']}
"""

    elif intent == "topic_query":
        topic_id = extract_topic(user_question)
        topic = db.get_topic(topic_id)
        related_stocks = db.get_topic_stocks(topic_id)

        context = f"""
## 使用者查詢題材：{topic['name']}

### 熱度資訊
- 當前熱度：{topic['heat_score']}°
- 趨勢：{topic['heat_trend']}
- 持續天數：{(today - topic['start_date']).days} 天

### 核心催化劑
{topic['catalysts']}

### 供應鏈股票（按 tier 分層）
{topic['supply_chain']}

### 投資策略
{topic['investment_strategy']}

### 避開清單
{topic['avoid_list']}
"""

    elif intent == "market_view":
        market_data = db.get_market_summary()
        hot_topics = db.get_top_topics(limit=5)

        context = f"""
## 當前市場狀況

### 大盤
{market_data}

### Top 5 熱門題材
{hot_topics}

### 市場警訊
{market_data['warnings']}
"""

    return context
```
